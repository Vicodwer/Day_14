# SQL Foundations Assignment

## Overview

This assignment demonstrates fundamental SQL operations used in relational databases.
The goal is to perform data retrieval, filtering, aggregation, and table joins while also translating SQL logic into equivalent **Pandas operations**.

This highlights the relationship between **database querying (SQL)** and **Python-based data analysis (Pandas)**.

---

## Learning Objectives

* Write structured SQL queries
* Understand relational database operations
* Use aggregation and grouping functions
* Implement table joins
* Translate SQL queries into Pandas equivalents

## SQL Concepts Implemented

* SELECT statements
* Filtering using WHERE
* Sorting using ORDER BY
* Aggregation functions (COUNT, SUM, AVG)
* GROUP BY and HAVING
* Table joins (INNER, LEFT, RIGHT)
* Subqueries

---

## Example Tasks

* Retrieve employees with salary greater than a threshold
* Calculate average salary per department
* Identify top earning employees
* Join employee and department datasets

Each SQL query is paired with a **Pandas equivalent operation** producing the same result.

---

## Key Takeaways

* SQL is optimized for database querying
* Pandas is powerful for in-memory data analysis
* Understanding both tools enables flexible data workflows
