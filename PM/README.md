# Advanced SQL: Window Functions, Subqueries and CTEs

## Overview

This assignment explores advanced SQL techniques commonly used in **data engineering and analytics workflows**.
The implementation focuses on window functions, subqueries, Common Table Expressions (CTEs), and recursive queries.

These techniques are frequently used in **data pipelines, business analytics, and interview scenarios for data engineering roles**.

---

## Learning Objectives

* Implement window functions for advanced analytics
* Use subqueries and correlated queries
* Create and utilize Common Table Expressions (CTEs)
* Understand recursive CTEs
* Optimize queries using window functions instead of correlated subqueries
